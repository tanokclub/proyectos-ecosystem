# CopiloC

Copiloto de código para equipos: indexa el repo, genera PRs, hace code review.

**Categoría:** AI/GenAI

## Stack

- Next.js 14
- NestJS
- Tree-sitter
- PostgreSQL
- Anthropic

## Módulos MVP

- indexing
- autocompletar
- PR gen
- review
- analytics

## Arranque

```bash
npm run dev       # Mock API en :4204
npm run dev:web   # Web dev en :3204
```

## Validar

```bash
npm test
```
