# AgriSust

Agritech sustentable: precision farming, agua, suelo, agricultura regenerativa.

**Categoría:** ClimateTech

## Stack

- Next.js 14
- FastAPI
- PostgreSQL
- LoRaWAN
- Anthropic

## Módulos MVP

- parcelas
- sensores
- riego
- recomendaciones
- mercado

## Arranque

```bash
npm run dev       # Mock API en :4235
npm run dev:web   # Web dev en :3235
```

## Validar

```bash
npm test
```
