# RAGForge

Plataforma de RAG empresarial: ingesta, embeddings, índices, evaluación.

**Categoría:** AI/GenAI

## Stack

- Next.js 14
- FastAPI
- pgvector
- Redis
- OpenAI

## Módulos MVP

- ingesta de documentos
- embeddings
- búsqueda híbrida
- chat con citas
- evaluación

## Arranque

```bash
npm run dev       # Mock API en :4200
npm run dev:web   # Web dev en :3200
```

## Validar

```bash
npm test
```
