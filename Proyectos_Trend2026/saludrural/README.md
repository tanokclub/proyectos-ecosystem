# SaludRural

IoT salud rural: telemetría, telediagnóstico, brigadas, conectividad satelital.

**Categoría:** IoT / Smart Cities

## Stack

- Next.js 14
- NestJS
- PostgreSQL
- Starlink
- Twilio

## Módulos MVP

- puestos rurales
- telediagnóstico
- inventario meds
- brigadas
- epidemiología

## Arranque

```bash
npm run dev       # Mock API en :4278
npm run dev:web   # Web dev en :3278
```

## Validar

```bash
npm test
```
