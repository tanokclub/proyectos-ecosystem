# PathfinderAI

Coaching de carrera con AI: roadmap personalizado y matching de oportunidades.

**Categoría:** EdTech

## Stack

- Next.js 14
- FastAPI
- PostgreSQL
- Anthropic

## Módulos MVP

- perfil
- roadmap
- oportunidades
- mentoring
- progreso

## Arranque

```bash
npm run dev       # Mock API en :4241
npm run dev:web   # Web dev en :3241
```

## Validar

```bash
npm test
```
