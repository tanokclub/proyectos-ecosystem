# AireSensor

Red IoT de sensores de calidad de aire en ciudades LATAM con alertas.

**Categoría:** ClimateTech

## Stack

- Next.js 14
- FastAPI
- TimescaleDB
- MQTT
- Mapbox

## Módulos MVP

- sensores
- mapas
- alertas
- predicción
- API pública

## Arranque

```bash
npm run dev       # Mock API en :4232
npm run dev:web   # Web dev en :3232
```

## Validar

```bash
npm test
```
