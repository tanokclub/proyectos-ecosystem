# ESGReport

Auto-reporte ESG (TCFD, ISSB, GRI) con scoring AI y data lake.

**Categoría:** ClimateTech

## Stack

- Next.js 14
- NestJS
- PostgreSQL
- dbt
- Anthropic

## Módulos MVP

- frameworks
- data ingestion
- scoring
- reportes
- audit

## Arranque

```bash
npm run dev       # Mock API en :4239
npm run dev:web   # Web dev en :3239
```

## Validar

```bash
npm test
```
