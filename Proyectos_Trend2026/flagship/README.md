# FlagShip

Feature flags self-hosted: rollouts gradual, A/B, segmentación, SDKs multi-lenguaje.

**Categoría:** DevTools / Infra

## Stack

- Next.js 14
- NestJS
- PostgreSQL
- Redis
- TypeScript

## Módulos MVP

- flags
- environments
- rollouts
- A/B
- analytics

## Arranque

```bash
npm run dev       # Mock API en :4250
npm run dev:web   # Web dev en :3250
```

## Validar

```bash
npm test
```
