# TreasuryX

Tesorería corporativa LATAM multi-moneda con cash forecasting AI.

**Categoría:** FinTech LATAM

## Stack

- Next.js 14
- NestJS
- PostgreSQL
- ClickHouse
- Prophet

## Módulos MVP

- multi-moneda
- forecast
- FX hedging
- pagos
- reportes

## Arranque

```bash
npm run dev       # Mock API en :4216
npm run dev:web   # Web dev en :3216
```

## Validar

```bash
npm test
```
