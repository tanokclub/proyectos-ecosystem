# RealEstateChain

Fractionalize real estate: tokens ERC-3643, rentas on-chain, marketplace LATAM.

**Categoría:** Web3 / Blockchain

## Stack

- Next.js 14
- Foundry
- Solidity
- Ethers
- PostgreSQL

## Módulos MVP

- propiedades
- tokenización
- rentas
- governance
- mercado secundario

## Arranque

```bash
npm run dev       # Mock API en :4264
npm run dev:web   # Web dev en :3264
```

## Validar

```bash
npm test
```
