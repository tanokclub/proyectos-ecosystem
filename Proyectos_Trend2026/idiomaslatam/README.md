# IdiomasLATAM

Aprende idiomas nativos LATAM (quechua, guaraní, náhuatl, aymara, mapudungun).

**Categoría:** EdTech

## Stack

- Next.js 14
- FastAPI
- PostgreSQL
- Whisper
- TTS

## Módulos MVP

- lecciones
- audio nativo
- speaking
- cultura
- comunidad

## Arranque

```bash
npm run dev       # Mock API en :4245
npm run dev:web   # Web dev en :3245
```

## Validar

```bash
npm test
```
