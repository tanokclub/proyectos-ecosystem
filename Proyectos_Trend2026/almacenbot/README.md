# AlmacenBot

Robotics warehouse management: AGVs, picking AI, inventario, WMS.

**Categoría:** IoT / Smart Cities

## Stack

- Next.js 14
- ROS2
- PostgreSQL
- Computer Vision
- Kafka

## Módulos MVP

- robots
- picking
- inventario
- WMS
- analytics

## Arranque

```bash
npm run dev       # Mock API en :4276
npm run dev:web   # Web dev en :3276
```

## Validar

```bash
npm test
```
