# SupplyChain

Trazabilidad supply chain on-chain: hashing por lote, IoT integrado, audit.

**Categoría:** Web3 / Blockchain

## Stack

- Next.js 14
- Hyperledger
- PostgreSQL
- Redis
- IoT

## Módulos MVP

- lotes
- trazabilidad
- sensores
- audit
- compliance

## Arranque

```bash
npm run dev       # Mock API en :4268
npm run dev:web   # Web dev en :3268
```

## Validar

```bash
npm test
```
