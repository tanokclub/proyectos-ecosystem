# CarbonMarket — Plan maestro

Marketplace de créditos de carbono verificados (Verra, Gold Standard) LATAM.

## Categoría

ClimateTech

## Módulos MVP

- catálogo
- proyectos
- compra
- retiro
- verificación

## Stack objetivo

- Next.js 14
- NestJS
- PostgreSQL
- Ethers
- Stripe

## Endpoints mock (8)

- `GET /api/v1/projects` — Proyectos
- `GET /api/v1/portfolio` — Portafolio
- `POST /api/v1/buy` — Comprar
- `POST /api/v1/retire` — Retirar
- `GET /api/v1/metrics` — Métricas del servicio
- `GET /api/v1/notifications` — Notificaciones del usuario
- `POST /api/v1/search` — Búsqueda global
- `POST /api/v1/batch` — Operación batch
