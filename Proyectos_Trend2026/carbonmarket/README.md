# CarbonMarket

Marketplace de créditos de carbono verificados (Verra, Gold Standard) LATAM.

**Categoría:** ClimateTech

## Stack

- Next.js 14
- NestJS
- PostgreSQL
- Ethers
- Stripe

## Módulos MVP

- catálogo
- proyectos
- compra
- retiro
- verificación

## Arranque

```bash
npm run dev       # Mock API en :4231
npm run dev:web   # Web dev en :3231
```

## Validar

```bash
npm test
```
