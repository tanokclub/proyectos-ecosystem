# CursosLive

Streaming educativo bidireccional: clases live, Q&A en vivo, certificación.

**Categoría:** EdTech

## Stack

- Next.js 14
- NestJS
- PostgreSQL
- WebRTC
- Redis

## Módulos MVP

- clases live
- salas
- Q&A
- certificación
- pagos

## Arranque

```bash
npm run dev       # Mock API en :4242
npm run dev:web   # Web dev en :3242
```

## Validar

```bash
npm test
```
