# GenomaLab

Registro genómico personal con interpretación AI y privacidad zero-knowledge.

**Categoría:** HealthTech

## Stack

- Next.js 14
- FastAPI
- PostgreSQL
- Snakemake
- IPFS

## Módulos MVP

- uploads
- pipeline
- variantes
- reportes
- sharing

## Arranque

```bash
npm run dev       # Mock API en :4227
npm run dev:web   # Web dev en :3227
```

## Validar

```bash
npm test
```
