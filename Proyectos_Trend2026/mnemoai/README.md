# MnemoAI

Banco de memoria persistente para agentes: hechos, decisiones, embeddings.

**Categoría:** AI/GenAI

## Stack

- Next.js 14
- FastAPI
- pgvector
- PostgreSQL
- Redis

## Módulos MVP

- hechos
- episodios
- recuperación
- olvido
- sync agentes

## Arranque

```bash
npm run dev       # Mock API en :4209
npm run dev:web   # Web dev en :3209
```

## Validar

```bash
npm test
```
