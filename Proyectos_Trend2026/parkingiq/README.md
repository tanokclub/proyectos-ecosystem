# ParkingIQ

Parking inteligente: detección plazas libres, pricing dinámico, reservas.

**Categoría:** IoT / Smart Cities

## Stack

- Next.js 14
- NestJS
- PostgreSQL
- Computer Vision
- Redis

## Módulos MVP

- plazas
- pricing
- reservas
- pagos
- analytics

## Arranque

```bash
npm run dev       # Mock API en :4271
npm run dev:web   # Web dev en :3271
```

## Validar

```bash
npm test
```
