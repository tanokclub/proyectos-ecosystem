# Sherpa AI

Agente de soporte multicanal (web, WhatsApp, email) con tools y handoff humano.

**Categoría:** AI/GenAI

## Stack

- Next.js 14
- NestJS
- PostgreSQL
- Redis
- Anthropic

## Módulos MVP

- tickets
- agente con tools
- handoff
- KB
- analytics

## Arranque

```bash
npm run dev       # Mock API en :4201
npm run dev:web   # Web dev en :3201
```

## Validar

```bash
npm test
```
