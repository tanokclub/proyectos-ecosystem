# PayHub

Super-app de pagos: P2P, QR, recargas, servicios, cashback.

**Categoría:** FinTech LATAM

## Stack

- Next.js 14
- NestJS
- PostgreSQL
- Redis
- Kafka

## Módulos MVP

- wallet
- P2P
- QR
- recargas
- cashback

## Arranque

```bash
npm run dev       # Mock API en :4219
npm run dev:web   # Web dev en :3219
```

## Validar

```bash
npm test
```
