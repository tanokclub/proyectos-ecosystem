# NewsletterX

Newsletter SaaS: editor, monetización, segmentación, A/B, analytics avanzados.

**Categoría:** Creator Economy

## Stack

- Next.js 14
- NestJS
- PostgreSQL
- SendGrid
- ClickHouse

## Módulos MVP

- editor
- envíos
- segmentación
- A/B
- monetización

## Arranque

```bash
npm run dev       # Mock API en :4282
npm run dev:web   # Web dev en :3282
```

## Validar

```bash
npm test
```
