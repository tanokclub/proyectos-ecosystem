# EVHub

Red de carga EV LATAM con roaming, ruta inteligente y reservas.

**Categoría:** ClimateTech

## Stack

- Next.js 14
- NestJS
- PostgreSQL
- OCPP
- Mapbox

## Módulos MVP

- estaciones
- roaming
- reservas
- ruta inteligente
- pagos

## Arranque

```bash
npm run dev       # Mock API en :4238
npm run dev:web   # Web dev en :3238
```

## Validar

```bash
npm test
```
