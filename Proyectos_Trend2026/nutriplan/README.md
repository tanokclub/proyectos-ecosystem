# NutriPlan

Nutrición AI con menús LATAM, conteo macros y compras integradas.

**Categoría:** HealthTech

## Stack

- Next.js 14
- FastAPI
- PostgreSQL
- Anthropic
- Rappi API

## Módulos MVP

- perfil
- menús
- recetas
- compras
- tracking

## Arranque

```bash
npm run dev       # Mock API en :4226
npm run dev:web   # Web dev en :3226
```

## Validar

```bash
npm test
```
