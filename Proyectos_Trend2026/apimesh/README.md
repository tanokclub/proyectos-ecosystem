# APIMesh

API gateway open source: routing, auth, rate limit, transformaciones, observabilidad.

**Categoría:** DevTools / Infra

## Stack

- Next.js 14
- Envoy
- Lua
- Redis
- PostgreSQL

## Módulos MVP

- routes
- auth
- rate limit
- transforms
- plugins

## Arranque

```bash
npm run dev       # Mock API en :4254
npm run dev:web   # Web dev en :3254
```

## Validar

```bash
npm test
```
