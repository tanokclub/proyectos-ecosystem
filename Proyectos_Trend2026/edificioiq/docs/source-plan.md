# EdificioIQ — Plan maestro

Building automation: HVAC, energía, ocupación, accesos, ESG en edificios.

## Categoría

IoT / Smart Cities

## Módulos MVP

- HVAC
- energía
- ocupación
- accesos
- reportes ESG

## Stack objetivo

- Next.js 14
- NestJS
- PostgreSQL
- BACnet
- TimescaleDB

## Endpoints mock (8)

- `GET /api/v1/buildings` — Edificios
- `GET /api/v1/energy` — Energía
- `POST /api/v1/hvac` — Setear HVAC
- `POST /api/v1/access` — Grant acceso
- `GET /api/v1/metrics` — Métricas del servicio
- `GET /api/v1/notifications` — Notificaciones del usuario
- `POST /api/v1/search` — Búsqueda global
- `POST /api/v1/batch` — Operación batch
