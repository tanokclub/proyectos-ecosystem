# EdificioIQ

Building automation: HVAC, energía, ocupación, accesos, ESG en edificios.

**Categoría:** IoT / Smart Cities

## Stack

- Next.js 14
- NestJS
- PostgreSQL
- BACnet
- TimescaleDB

## Módulos MVP

- HVAC
- energía
- ocupación
- accesos
- reportes ESG

## Arranque

```bash
npm run dev       # Mock API en :4279
npm run dev:web   # Web dev en :3279
```

## Validar

```bash
npm test
```
