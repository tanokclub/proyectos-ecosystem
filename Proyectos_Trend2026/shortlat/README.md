# ShortLAT

Plataforma short video estilo TikTok con monetización LATAM y AI recommend.

**Categoría:** Creator Economy

## Stack

- Next.js 14
- NestJS
- PostgreSQL
- CDN
- Redis

## Módulos MVP

- feed
- subida
- editor
- monetización
- creator fund

## Arranque

```bash
npm run dev       # Mock API en :4283
npm run dev:web   # Web dev en :3283
```

## Validar

```bash
npm test
```
