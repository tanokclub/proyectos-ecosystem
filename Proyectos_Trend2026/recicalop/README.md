# ReciclaLoop

Plataforma de economía circular: residuos B2B + marketplace materiales reciclados.

**Categoría:** ClimateTech

## Stack

- Next.js 14
- NestJS
- PostgreSQL
- Redis
- Stripe

## Módulos MVP

- oferentes
- demandantes
- logística
- trazabilidad
- reportes

## Arranque

```bash
npm run dev       # Mock API en :4236
npm run dev:web   # Web dev en :3236
```

## Validar

```bash
npm test
```
