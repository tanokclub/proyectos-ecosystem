# CitySense

Smart city dashboard: sensores urbanos, alertas, gestión incidentes municipales.

**Categoría:** IoT / Smart Cities

## Stack

- Next.js 14
- FastAPI
- PostgreSQL
- TimescaleDB
- MQTT

## Módulos MVP

- sensores urbanos
- incidentes
- dashboard
- predicción
- ciudadanos

## Arranque

```bash
npm run dev       # Mock API en :4270
npm run dev:web   # Web dev en :3270
```

## Validar

```bash
npm test
```
