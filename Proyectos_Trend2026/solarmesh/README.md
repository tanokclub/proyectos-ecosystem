# SolarMesh

Plataforma para microgrids solares comunitarios con tokenización energía.

**Categoría:** ClimateTech

## Stack

- Next.js 14
- NestJS
- PostgreSQL
- InfluxDB
- Ethers

## Módulos MVP

- sites
- producción
- consumo
- P2P energy
- tokens

## Arranque

```bash
npm run dev       # Mock API en :4233
npm run dev:web   # Web dev en :3233
```

## Validar

```bash
npm test
```
