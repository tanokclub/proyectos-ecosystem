# CarbonTrack

Medición de huella de carbono corporativa scope 1/2/3 con auto-reporte GHG.

**Categoría:** ClimateTech

## Stack

- Next.js 14
- NestJS
- PostgreSQL
- dbt
- Redis

## Módulos MVP

- inventario
- scope 1/2/3
- reportes
- GHG protocol
- sci targets

## Arranque

```bash
npm run dev       # Mock API en :4230
npm run dev:web   # Web dev en :3230
```

## Validar

```bash
npm test
```
