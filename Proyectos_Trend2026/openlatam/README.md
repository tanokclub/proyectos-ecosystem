# OpenLatAm

Open banking LATAM: agrega cuentas vía PIX, CoDi, PSE, CMF, CMA.

**Categoría:** FinTech LATAM

## Stack

- Next.js 14
- NestJS
- PostgreSQL
- Redis
- Kafka

## Módulos MVP

- conexiones
- cuentas
- transacciones
- consents
- agregación

## Arranque

```bash
npm run dev       # Mock API en :4210
npm run dev:web   # Web dev en :3210
```

## Validar

```bash
npm test
```
