# ReforestaApp

Tracker satelital de proyectos de reforestación con verificación geoespacial.

**Categoría:** ClimateTech

## Stack

- Next.js 14
- FastAPI
- PostGIS
- Sentinel-2
- Anthropic

## Módulos MVP

- proyectos
- parcelas
- monitoreo NDVI
- verificación
- reportes

## Arranque

```bash
npm run dev       # Mock API en :4234
npm run dev:web   # Web dev en :3234
```

## Validar

```bash
npm test
```
