# FanClub

Suscripciones fan: tiers, contenido exclusivo, comunidad, eventos virtuales.

**Categoría:** Creator Economy

## Stack

- Next.js 14
- NestJS
- PostgreSQL
- Stripe
- WebRTC

## Módulos MVP

- tiers
- contenido
- comunidad
- eventos
- mensajería

## Arranque

```bash
npm run dev       # Mock API en :4285
npm run dev:web   # Web dev en :3285
```

## Validar

```bash
npm test
```
