# CIChain

CI/CD self-hosted compatible con GitHub Actions, ejecutores en cualquier nube.

**Categoría:** DevTools / Infra

## Stack

- Next.js 14
- Go
- PostgreSQL
- Docker
- Redis

## Módulos MVP

- workflows
- runners
- artifacts
- secrets
- matrix

## Arranque

```bash
npm run dev       # Mock API en :4255
npm run dev:web   # Web dev en :3255
```

## Validar

```bash
npm test
```
