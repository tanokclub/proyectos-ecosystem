# MusicSplit

Royalty splits para músicos: smart contracts, distribución automática, regalías globales.

**Categoría:** Creator Economy

## Stack

- Next.js 14
- Foundry
- PostgreSQL
- Ethers
- Spotify API

## Módulos MVP

- canciones
- splits
- distribución
- cobros
- analytics

## Arranque

```bash
npm run dev       # Mock API en :4287
npm run dev:web   # Web dev en :3287
```

## Validar

```bash
npm test
```
