# SecretsVault

Gestión de secretos zero-trust con rotación automática y audit total.

**Categoría:** DevTools / Infra

## Stack

- Next.js 14
- NestJS
- PostgreSQL
- Vault
- KMS

## Módulos MVP

- secretos
- rotación
- políticas
- audit
- SDKs

## Arranque

```bash
npm run dev       # Mock API en :4253
npm run dev:web   # Web dev en :3253
```

## Validar

```bash
npm test
```
