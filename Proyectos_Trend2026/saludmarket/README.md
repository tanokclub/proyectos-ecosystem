# SaludMarket

Marketplace de servicios de salud: estudios, vacunas, terapias a domicilio.

**Categoría:** HealthTech

## Stack

- Next.js 14
- NestJS
- PostgreSQL
- Redis
- Stripe

## Módulos MVP

- catálogo
- agenda
- pagos
- proveedores
- reviews

## Arranque

```bash
npm run dev       # Mock API en :4229
npm run dev:web   # Web dev en :3229
```

## Validar

```bash
npm test
```
