# SkillBoard

Recruiting con skill graph: matching técnico, evaluación, take-home automatizado.

**Categoría:** EdTech

## Stack

- Next.js 14
- NestJS
- PostgreSQL
- XGBoost
- Docker

## Módulos MVP

- perfil técnico
- jobs
- matching
- evaluaciones
- analytics

## Arranque

```bash
npm run dev       # Mock API en :4248
npm run dev:web   # Web dev en :3248
```

## Validar

```bash
npm test
```
