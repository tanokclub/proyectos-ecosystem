# KidsAcad

Plataforma educativa para niños 5-12: gamification, parental control, currículo LATAM.

**Categoría:** EdTech

## Stack

- Next.js 14
- NestJS
- PostgreSQL
- Phaser
- Firebase

## Módulos MVP

- actividades
- gamification
- parental
- progreso
- reportes

## Arranque

```bash
npm run dev       # Mock API en :4246
npm run dev:web   # Web dev en :3246
```

## Validar

```bash
npm test
```
