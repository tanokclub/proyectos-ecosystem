# TokenLaunch

Launchpad de tokens: IDO, LBP, vesting, anti-bot, KYC integrado.

**Categoría:** Web3 / Blockchain

## Stack

- Next.js 14
- Foundry
- Solidity
- Ethers
- PostgreSQL

## Módulos MVP

- proyectos
- IDO
- vesting
- KYC
- staking tiers

## Arranque

```bash
npm run dev       # Mock API en :4262
npm run dev:web   # Web dev en :3262
```

## Validar

npm test
