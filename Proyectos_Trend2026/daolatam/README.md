# DAOLatAm

Gobernanza DAO LATAM: propuestas, votación on-chain, treasury, snapshots.

**Categoría:** Web3 / Blockchain

## Stack

- Next.js 14
- Foundry
- Solidity
- Ethers
- PostgreSQL

## Módulos MVP

- propuestas
- votación
- treasury
- snapshots
- delegación

## Arranque

```bash
npm run dev       # Mock API en :4260
npm run dev:web   # Web dev en :3260
```

## Validar

```bash
npm test
```
