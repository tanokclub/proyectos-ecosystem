# ChaosTest

Chaos engineering as service: fault injection, game days, hipótesis automatizadas.

**Categoría:** DevTools / Infra

## Stack

- Next.js 14
- Go
- PostgreSQL
- eBPF
- Kubernetes

## Módulos MVP

- experimentos
- fault injection
- hypothesis
- reportes
- game days
