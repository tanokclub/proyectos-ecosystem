# TestPrep

Preparación adaptativa a exámenes oficiales: LSAT, MCAT, ICFES, ENEM, PAES.

**Categoría:** EdTech

## Stack

- Next.js 14
- FastAPI
- PostgreSQL
- XGBoost
- Anthropic

## Módulos MVP

- simulacros
- banco preguntas
- adaptive learning
- analytics
- tutor

## Arranque

```bash
npm run dev       # Mock API en :4243
npm run dev:web   # Web dev en :3243
```

## Validar

```bash
npm test
```
