# TriajeAI

Triage AI para urgencias y atención primaria con clasificación Manchester.

**Categoría:** HealthTech

## Stack

- Next.js 14
- FastAPI
- PostgreSQL
- Anthropic

## Módulos MVP

- triage
- protocolos
- derivación
- audit
- analytics

## Arranque

```bash
npm run dev       # Mock API en :4222
npm run dev:web   # Web dev en :3222
```

## Validar

```bash
npm test
```
