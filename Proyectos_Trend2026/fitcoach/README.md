# FitCoach

Entrenador AI con plan personalizado: ejercicio, recuperación, nutrición.

**Categoría:** HealthTech

## Stack

- Next.js 14
- FastAPI
- PostgreSQL
- Anthropic

## Módulos MVP

- plan
- rutinas
- wearable sync
- progreso
- comunidad

## Arranque

```bash
npm run dev       # Mock API en :4225
npm run dev:web   # Web dev en :3225
```

## Validar

```bash
npm test
```
