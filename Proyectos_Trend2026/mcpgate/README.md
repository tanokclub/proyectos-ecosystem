# MCPGate

Gestor de servidores MCP: descubrir, instalar, monitorizar y autorizar.

**Categoría:** AI/GenAI

## Stack

- Next.js 14
- NestJS
- PostgreSQL
- Redis
- Docker

## Módulos MVP

- catálogo
- instalación
- autenticación
- logs
- permisos

## Arranque

```bash
npm run dev       # Mock API en :4206
npm run dev:web   # Web dev en :3206
```

## Validar

```bash
npm test
```
