# ObservaLAT

Observabilidad open source: logs, métricas, traces, alertas, OpenTelemetry.

**Categoría:** DevTools / Infra

## Stack

- Next.js 14
- Go
- ClickHouse
- OpenTelemetry
- Redis

## Módulos MVP

- logs
- métricas
- traces
- alertas
- dashboards

## Arranque

```bash
npm run dev       # Mock API en :4251
npm run dev:web   # Web dev en :3251
```

## Validar

npm test
