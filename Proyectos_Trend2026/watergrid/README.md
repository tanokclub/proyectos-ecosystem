# WaterGrid

Gestión hídrica urbana: medición fugas, calidad, presión, alertas en tiempo real.

**Categoría:** IoT / Smart Cities

## Stack

- Next.js 14
- FastAPI
- TimescaleDB
- MQTT
- GIS

## Módulos MVP

- sensores
- fugas
- calidad
- consumo
- alertas

## Arranque

```bash
npm run dev       # Mock API en :4272
npm run dev:web   # Web dev en :3272
```

## Validar

```bash
npm test
```
