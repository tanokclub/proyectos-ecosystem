# PodcastLAT

Red de podcasts LATAM: hosting, monetización, analytics, transcripción AI.

**Categoría:** Creator Economy

## Stack

- Next.js 14
- NestJS
- PostgreSQL
- Whisper
- CDN

## Módulos MVP

- shows
- episodios
- distribución
- monetización
- transcripción

## Arranque

```bash
npm run dev       # Mock API en :4281
npm run dev:web   # Web dev en :3281
```

## Validar

```bash
npm test
```
