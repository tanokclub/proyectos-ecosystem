# WasteSmart

Recolección residuos optimizada: sensores en contenedores, rutas dinámicas.

**Categoría:** IoT / Smart Cities

## Stack

- Next.js 14
- NestJS
- PostgreSQL
- OR-Tools
- MQTT

## Módulos MVP

- contenedores
- rutas
- fleet
- reportes ciudadanos
- analytics

## Arranque

```bash
npm run dev       # Mock API en :4273
npm run dev:web   # Web dev en :3273
```

## Validar

```bash
npm test
```
