# PromptOps

Versionado, testing y deploy de prompts en producción con métricas.

**Categoría:** AI/GenAI

## Stack

- Next.js 14
- NestJS
- PostgreSQL
- ClickHouse
- Anthropic

## Módulos MVP

- versiones
- A/B testing
- eval
- observabilidad
- rollback

## Arranque

```bash
npm run dev       # Mock API en :4208
npm run dev:web   # Web dev en :3208
```

## Validar

```bash
npm test
```
