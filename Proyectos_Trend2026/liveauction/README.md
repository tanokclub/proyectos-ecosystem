# LiveAuction

Livestream commerce: subastas en vivo, ventas flash, gamification, multi-creator.

**Categoría:** Creator Economy

## Stack

- Next.js 14
- NestJS
- PostgreSQL
- WebRTC
- Stripe

## Módulos MVP

- streams
- subastas
- productos
- checkout
- analytics

## Arranque

```bash
npm run dev       # Mock API en :4284
npm run dev:web   # Web dev en :3284
```

## Validar

```bash
npm test
```
