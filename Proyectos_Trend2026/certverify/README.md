# CertVerify

Verificación blockchain de títulos académicos y certificaciones profesionales.

**Categoría:** EdTech

## Stack

- Next.js 14
- NestJS
- PostgreSQL
- Ethers
- IPFS

## Módulos MVP

- emisión
- verificación
- wallet credenciales
- integraciones
- audit

## Arranque

```bash
npm run dev       # Mock API en :4247
npm run dev:web   # Web dev en :3247
```

## Validar

```bash
npm test
```
