# FlotaTrack

Fleet management LATAM: GPS, ELD, mantenimiento predictivo, telemática.

**Categoría:** IoT / Smart Cities

## Stack

- Next.js 14
- NestJS
- PostgreSQL
- TimescaleDB
- OBD-II

## Módulos MVP

- vehículos
- telemática
- rutas
- mantenimiento
- compliance

## Arranque

```bash
npm run dev       # Mock API en :4275
npm run dev:web   # Web dev en :3275
```

## Validar

```bash
npm test
```
