# SendyLat

Remesas LATAM-to-LATAM con rails locales y FX competitivo.

**Categoría:** FinTech LATAM

## Stack

- Next.js 14
- NestJS
- PostgreSQL
- Redis
- Stellar

## Módulos MVP

- envío
- pickup
- KYC
- FX
- compliance

## Arranque

```bash
npm run dev       # Mock API en :4212
npm run dev:web   # Web dev en :3212
```

## Validar

```bash
npm test
```
