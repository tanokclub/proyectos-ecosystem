# TelemedLAT

Telemedicina LATAM: consultas video, recetas digitales, integración EPS.

**Categoría:** HealthTech

## Stack

- Next.js 14
- NestJS
- PostgreSQL
- Twilio Video
- Redis

## Módulos MVP

- agenda
- videoconsulta
- recetas
- historial
- pagos

## Arranque

```bash
npm run dev       # Mock API en :4220
npm run dev:web   # Web dev en :3220
```

## Validar

```bash
npm test
```
