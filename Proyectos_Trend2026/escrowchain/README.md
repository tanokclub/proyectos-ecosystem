# EscrowChain

Escrow on-chain para freelance, M&A y comercio internacional.

**Categoría:** FinTech LATAM

## Stack

- Next.js 14
- Foundry
- Solidity
- PostgreSQL
- Ethers

## Módulos MVP

- contratos
- milestones
- disputas
- oracle
- multisig

## Arranque

```bash
npm run dev       # Mock API en :4218
npm run dev:web   # Web dev en :3218
```

## Validar

```bash
npm test
```
