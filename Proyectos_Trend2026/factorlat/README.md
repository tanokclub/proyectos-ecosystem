# FactorLAT

Marketplace de factoring/invoice finance B2B en LATAM.

**Categoría:** FinTech LATAM

## Stack

- Next.js 14
- NestJS
- PostgreSQL
- Redis
- Postgres

## Módulos MVP

- emisores
- inversionistas
- subasta
- cobranza
- compliance

## Arranque

```bash
npm run dev       # Mock API en :4217
npm run dev:web   # Web dev en :3217
```

## Validar

```bash
npm test
```
