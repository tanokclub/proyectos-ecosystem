# CreadorPro

Plataforma all-in-one monetización creators: suscripciones, tips, comisiones, sponsors.

**Categoría:** Creator Economy

## Stack

- Next.js 14
- NestJS
- PostgreSQL
- Stripe Connect
- Redis

## Módulos MVP

- perfil
- suscripciones
- tips
- comisiones
- sponsors

## Arranque

```bash
npm run dev       # Mock API en :4280
npm run dev:web   # Web dev en :3280
```

## Validar

```bash
npm test
```
