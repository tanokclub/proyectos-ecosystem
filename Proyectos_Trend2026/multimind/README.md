# Multimind

Chatbot multimodal: texto, voz, imagen y video como entradas y salidas.

**Categoría:** AI/GenAI

## Stack

- Next.js 14
- FastAPI
- PostgreSQL
- Whisper
- SD3

## Módulos MVP

- chat
- visión
- audio
- video
- historial

## Arranque

```bash
npm run dev       # Mock API en :4203
npm run dev:web   # Web dev en :3203
```

## Validar

```bash
npm test
```
