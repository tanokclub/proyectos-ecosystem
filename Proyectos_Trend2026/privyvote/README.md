# PrivyVote

Voting privado ZK para DAOs, organizaciones y gobiernos: anónimo y verificable.

**Categoría:** Web3 / Blockchain

## Stack

- Next.js 14
- Foundry
- Circom
- Ethers
- PostgreSQL

## Módulos MVP

- proposals
- voting ZK
- tally
- audit
- verifiers

## Arranque

```bash
npm run dev       # Mock API en :4267
npm run dev:web   # Web dev en :3267
```

## Validar

```bash
npm test
```
