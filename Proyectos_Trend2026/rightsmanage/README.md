# RightsManage

Gestión IP y derechos: catálogo, licenciamientos, takedowns, sync rights.

**Categoría:** Creator Economy

## Stack

- Next.js 14
- NestJS
- PostgreSQL
- Elastic
- CMS

## Módulos MVP

- catálogo IP
- licencias
- takedowns
- sync rights
- analytics

## Arranque

```bash
npm run dev       # Mock API en :4289
npm run dev:web   # Web dev en :3289
```

## Validar

```bash
npm test
```
