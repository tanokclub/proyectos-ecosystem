# CompudOps

Orchestrator de agentes computer-use: lanza, monitoriza y graba sesiones.

**Categoría:** AI/GenAI

## Stack

- Next.js 14
- FastAPI
- PostgreSQL
- noVNC
- Anthropic

## Módulos MVP

- sesiones
- tareas
- video
- aprobaciones
- audit

## Arranque

```bash
npm run dev       # Mock API en :4207
npm run dev:web   # Web dev en :3207
```

## Validar

```bash
npm test
```
