# IdentityID

Identidad SSI/DID LATAM: credenciales verificables, presentaciones zero-knowledge.

**Categoría:** Web3 / Blockchain

## Stack

- Next.js 14
- NestJS
- PostgreSQL
- DID
- ZK-SNARK

## Módulos MVP

- wallet ID
- credenciales
- presentaciones
- verificadores
- recovery

## Arranque

```bash
npm run dev       # Mock API en :4265
npm run dev:web   # Web dev en :3265
```

## Validar

```bash
npm test
```
