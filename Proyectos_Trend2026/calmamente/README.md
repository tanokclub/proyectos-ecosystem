# CalmaMente

Salud mental con chat AI + matching con terapeutas LATAM certificados.

**Categoría:** HealthTech

## Stack

- Next.js 14
- NestJS
- PostgreSQL
- Anthropic
- Twilio

## Módulos MVP

- chat
- screening
- matching
- sesiones
- crisis

## Arranque

```bash
npm run dev       # Mock API en :4224
npm run dev:web   # Web dev en :3224
```

## Validar

```bash
npm test
```
