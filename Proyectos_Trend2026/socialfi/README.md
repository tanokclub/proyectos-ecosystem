# SocialFi

Social tokens y economía creator: monetización on-chain, gates de contenido.

**Categoría:** Web3 / Blockchain

## Stack

- Next.js 14
- NestJS
- PostgreSQL
- Ethers
- Redis

## Módulos MVP

- tokens creator
- gating
- tips
- rewards
- marketplace

## Arranque

```bash
npm run dev       # Mock API en :4266
npm run dev:web   # Web dev en :3266
```

## Validar

```bash
npm test
```
