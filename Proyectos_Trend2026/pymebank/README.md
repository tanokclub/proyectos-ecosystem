# PymeBank

Neobanco para PYMEs LATAM: cuenta empresarial, facturación, nómina.

**Categoría:** FinTech LATAM

## Stack

- Next.js 14
- NestJS
- PostgreSQL
- Kafka
- Redis

## Módulos MVP

- cuenta
- facturación SAT/DIAN
- nómina
- préstamos
- tarjetas

## Arranque

```bash
npm run dev       # Mock API en :4214
npm run dev:web   # Web dev en :3214
```

## Validar

```bash
npm test
```
