# AIStudio

Content generation AI para creators: thumbnails, scripts, ediciones, voiceovers.

**Categoría:** Creator Economy

## Stack

- Next.js 14
- FastAPI
- PostgreSQL
- Anthropic
- SD3

## Módulos MVP

- thumbnails
- scripts
- ediciones
- voiceovers
- translaciones

## Arranque

```bash
npm run dev       # Mock API en :4288
npm run dev:web   # Web dev en :3288
```

## Validar

```bash
npm test
```
