# CharityChain

Donaciones transparentes on-chain con milestones y verificación de impacto.

**Categoría:** Web3 / Blockchain

## Stack

- Next.js 14
- Foundry
- Solidity
- Ethers
- IPFS

## Módulos MVP

- campañas
- milestones
- donaciones
- verificación impacto
- reportes

## Arranque

```bash
npm run dev       # Mock API en :4269
npm run dev:web   # Web dev en :3269
```

## Validar

```bash
npm test
```
