# CryptoLat

Wallet cripto LATAM con on/off-ramp local (PIX, SPEI, PSE).

**Categoría:** FinTech LATAM

## Stack

- Next.js 14
- NestJS
- PostgreSQL
- Ethers
- Bitcore

## Módulos MVP

- wallet
- on-ramp
- off-ramp
- swap
- staking

## Arranque

```bash
npm run dev       # Mock API en :4211
npm run dev:web   # Web dev en :3211
```

## Validar

```bash
npm test
```
