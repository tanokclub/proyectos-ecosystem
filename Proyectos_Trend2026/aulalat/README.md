# AulaLAT

LMS LATAM con AI tutor, learning paths adaptativos y multi-idioma.

**Categoría:** EdTech

## Stack

- Next.js 14
- NestJS
- PostgreSQL
- Redis
- Anthropic

## Módulos MVP

- cursos
- AI tutor
- evaluación
- gamification
- analytics

## Arranque

```bash
npm run dev       # Mock API en :4240
npm run dev:web   # Web dev en :3240
```

## Validar

```bash
npm test
```
