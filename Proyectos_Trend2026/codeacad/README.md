# CodeAcad

Bootcamp coding LATAM con proyectos reales, mentores y bolsa de empleo.

**Categoría:** EdTech

## Stack

- Next.js 14
- NestJS
- PostgreSQL
- Docker
- Redis

## Módulos MVP

- cohortes
- proyectos
- mentoring
- sandbox
- placement

## Arranque

```bash
npm run dev       # Mock API en :4244
npm run dev:web   # Web dev en :3244
```

## Validar

```bash
npm test
```
