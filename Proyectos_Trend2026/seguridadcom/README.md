# SeguridadCom

Vigilancia comunitaria: cámaras, IA detección, botón pánico, alertas vecinos.

**Categoría:** IoT / Smart Cities

## Stack

- Next.js 14
- NestJS
- PostgreSQL
- CV
- WebRTC

## Módulos MVP

- cámaras
- alertas
- botón pánico
- vecinos
- autoridades

## Arranque

```bash
npm run dev       # Mock API en :4274
npm run dev:web   # Web dev en :3274
```

## Validar

```bash
npm test
```
