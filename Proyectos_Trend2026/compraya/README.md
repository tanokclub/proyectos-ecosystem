# CompraYa

BNPL para retail LATAM: split en 3-12 cuotas sin intereses.

**Categoría:** FinTech LATAM

## Stack

- Next.js 14
- NestJS
- PostgreSQL
- Redis
- Stripe

## Módulos MVP

- checkout
- cuotas
- scoring
- cobranza
- merchants

## Arranque

```bash
npm run dev       # Mock API en :4213
npm run dev:web   # Web dev en :3213
```

## Validar

```bash
npm test
```
