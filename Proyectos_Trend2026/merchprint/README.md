# MerchPrint

Print-on-demand LATAM: catálogo merchandising, integración tiendas, dropshipping.

**Categoría:** Creator Economy

## Stack

- Next.js 14
- NestJS
- PostgreSQL
- Stripe
- Stripe Connect

## Módulos MVP

- catálogo
- tiendas
- pedidos
- producción
- envíos

## Arranque

```bash
npm run dev       # Mock API en :4286
npm run dev:web   # Web dev en :3286
```

## Validar

```bash
npm test
```
