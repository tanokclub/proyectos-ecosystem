# ClimaModel

Modelado climático local con descarga estadística y proyecciones IPCC.

**Categoría:** ClimateTech

## Stack

- Next.js 14
- FastAPI
- xarray
- PostgreSQL
- Anthropic

## Módulos MVP

- scenarios
- descarga
- ensemble
- riesgos
- reportes

## Arranque

```bash
npm run dev       # Mock API en :4237
npm run dev:web   # Web dev en :3237
```

## Validar

```bash
npm test
```
