# MedReminder

App de adherencia a tratamiento: recordatorios, refills, analytics.

**Categoría:** HealthTech

## Stack

- Next.js 14
- NestJS
- PostgreSQL
- Firebase
- Twilio

## Módulos MVP

- recordatorios
- medicamentos
- adherencia
- farmacia
- caregivers

## Arranque

```bash
npm run dev       # Mock API en :4223
npm run dev:web   # Web dev en :3223
```

## Validar

```bash
npm test
```
