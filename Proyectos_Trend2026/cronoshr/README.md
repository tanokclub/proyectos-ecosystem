# CronosHR

EHR open source LATAM: historial clínico unificado HL7/FHIR.

**Categoría:** HealthTech

## Stack

- Next.js 14
- NestJS
- PostgreSQL
- HAPI FHIR
- OAuth2

## Módulos MVP

- pacientes
- encounters
- condiciones
- medicación
- observaciones

## Arranque

```bash
npm run dev       # Mock API en :4221
npm run dev:web   # Web dev en :3221
```

## Validar

```bash
npm test
```
