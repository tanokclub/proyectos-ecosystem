# VoiceClone Pro

Estudio para clonar voces, generar narraciones y voiceovers multilingües.

**Categoría:** AI/GenAI

## Stack

- Next.js 14
- FastAPI
- ElevenLabs
- S3
- PostgreSQL

## Módulos MVP

- voces
- proyectos
- síntesis TTS
- studio
- API

## Arranque

```bash
npm run dev       # Mock API en :4202
npm run dev:web   # Web dev en :3202
```

## Validar

```bash
npm test
```
