# DeployHaus

PaaS bare-metal LATAM: deploy Git-based, autoscaling, secretos, dominios.

**Categoría:** DevTools / Infra

## Stack

- Next.js 14
- NestJS
- Docker
- Nomad
- PostgreSQL

## Módulos MVP

- apps
- builds
- autoscaling
- secretos
- dominios

## Arranque

```bash
npm run dev       # Mock API en :4252
npm run dev:web   # Web dev en :3252
```

## Validar

```bash
npm test
```
