# NFTArte — Plan maestro

Marketplace NFT arte LATAM con royalties on-chain y curaduría.

## Categoría

Web3 / Blockchain

## Módulos MVP

- catálogo
- subastas
- royalties
- curaduría
- minting

## Stack objetivo

- Next.js 14
- NestJS
- PostgreSQL
- Ethers
- IPFS

## Endpoints mock (8)

- `GET /api/v1/listings` — Listings
- `GET /api/v1/auctions` — Subastas
- `POST /api/v1/mint` — Mint
- `POST /api/v1/bid` — Bid
- `GET /api/v1/metrics` — Métricas del servicio
- `GET /api/v1/notifications` — Notificaciones del usuario
- `POST /api/v1/search` — Búsqueda global
- `POST /api/v1/batch` — Operación batch
