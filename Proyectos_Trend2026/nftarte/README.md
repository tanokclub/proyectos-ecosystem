# NFTArte

Marketplace NFT arte LATAM con royalties on-chain y curaduría.

**Categoría:** Web3 / Blockchain

## Stack

- Next.js 14
- NestJS
- PostgreSQL
- Ethers
- IPFS

## Módulos MVP

- catálogo
- subastas
- royalties
- curaduría
- minting

## Arranque

```bash
npm run dev       # Mock API en :4261
npm run dev:web   # Web dev en :3261
```

## Validar

```bash
npm test
```
