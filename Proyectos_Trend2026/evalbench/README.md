# EvalBench

Plataforma para evaluar y comparar modelos LLM con datasets y métricas custom.

**Categoría:** AI/GenAI

## Stack

- Next.js 14
- FastAPI
- PostgreSQL
- ClickHouse
- Polars

## Módulos MVP

- datasets
- runs
- métricas
- comparador
- reportes

## Arranque

```bash
npm run dev       # Mock API en :4205
npm run dev:web   # Web dev en :3205
```

## Validar

```bash
npm test
```
