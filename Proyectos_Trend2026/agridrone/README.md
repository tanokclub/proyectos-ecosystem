# AgriDrone

Drones agrícolas para mapeo, monitoreo cultivos y aspersión de precisión.

**Categoría:** IoT / Smart Cities

## Stack

- Next.js 14
- FastAPI
- PostgreSQL
- CV
- PX4

## Módulos MVP

- flotas drones
- misiones
- mapeo NDVI
- aspersión
- reportes

## Arranque

```bash
npm run dev       # Mock API en :4277
npm run dev:web   # Web dev en :3277
```

## Validar

```bash
npm test
```
